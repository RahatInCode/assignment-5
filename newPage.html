What are the different ways to select an element in the DOM?
Use document.createElement("tagName") to create a new HTML element in JavaScript. This method generates an element but does not add it to the page immediately. Adding an Element to the DOM To insert the created element into the page, use: appendChild() – Adds the element at the end of a parent. prepend() – Adds the element at the beginning. insertBefore() – Inserts before a specific element.
What is the difference between innerHTML, innerText, and textContent ?
innerHTML: Gets or sets HTML content, including tags. Parses and renders HTML. innerText: Gets or sets only visible text, ignoring hidden content and treating HTML as plain text. textContent: Gets or sets all text, including hidden elements, without parsing HTML.
What is event delegation in the DOM?
Event delegation is a technique in JavaScript where a parent element handles events for its child elements, even if those children are added dynamically. Instead of attaching event listeners to multiple elements, you add a single listener to a common ancestor.
What is event bubbling in the DOM?
Event bubbling is a process where an event starts from the target element and propagates up through its parent elements in the DOM hierarchy. How It Works When an event occurs on an element, it first runs its event handler. Then, the event moves up to its parent, triggering their handlers (if any). This continues until it reaches the root element.
